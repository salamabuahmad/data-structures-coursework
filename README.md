# Data Structures Coursework

This repository contains selected projects from a Data Structures course.

The projects focus on implementing and using core data structures such as AVL trees, hash tables, dynamic arrays, and Union-Find / Disjoint Set structures in C++.

## Projects

### WET01 - Pirate Ships with AVL Trees

A C++ system for managing ships and pirates using AVL trees.

The system supports adding/removing ships and pirates, updating pirate treasure, moving pirates between ships, querying the richest pirate on a ship, and simulating ship battles.

### WET02 - Pirate Fleets with Hash Table and Union-Find

A C++ system for managing pirate fleets using custom data structures.

The system uses a hash table, AVL trees, dynamic arrays, and Union-Find to support fleet creation, pirate assignment, fleet merging, salary updates, and pirate rank-based arguments.
